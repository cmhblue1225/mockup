📚 Book Search API

알라딘 OpenAPI를 활용한 책 검색 및 조회 기능을 제공합니다.

🔍 제목/저자/출판사로 책 검색
📖 ISBN으로 책 정보 조회 (바코드 스캔 지원)
🎯 통합 검색 (ISBN/제목 자동 판별)
📄 상세한 책 정보 (표지, 설명, 가격 등)

주요 엔드포인트

GET /api/books/search 제목/저자로 검색
GET /api/books/isbn/{isbn} ISBN으로 조회
GET /api/books/{bookId} 책 상세 정보
GET /api/books 통합 검색
📖 Swagger 문서 보기 💻 GitHub 레포지토리

기술 스택

Node.js Express TypeScript Swagger/OpenAPI Netlify Functions

빠른 시작 예제

JavaScript 
// 책 검색 예제
const searchBooks = async (query) => {
  const response = await fetch(
    `https://lucky-lollipop-31b7e2.netlify.app/apis/api/books/search?query=${query}`
  );
  const data = await response.json();
  return data;
};

// 사용 예시
searchBooks('클린 코드')
  .then(result => {
    console.log('검색 결과:', result.data.books);
  });
cURL 
curl "https://lucky-lollipop-31b7e2.netlify.app/apis/api/books/search?query=클린코드"
